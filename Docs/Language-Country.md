# Country & Language



## Country

ISO 3166

| Property | Type   | Definition            |
| -------- | ------ | --------------------- |
| country  | string | 2 letter country code |



## Language

BCP 47



| Property | Type   | Definition                           |
| -------- | ------ | ------------------------------------ |
| language | string | Language code, ,may include dialects |
