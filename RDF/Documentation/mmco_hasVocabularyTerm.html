label

range
Vocabulary Terms
definition
Identifies the formal vocabulary term used to represent the entity to which this is attached. For example, is there is an instance of AssetStructuralImage called "JPEGimage" then this property would have, as its value, the vocabulary term that repreent jpeg images. The instance "JPEGimage" would probably contain additional properties, which is why we distinguish it from the vocabulary term itself.
