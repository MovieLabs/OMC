definition
A virtual machine is typically an instantiation of a single workspace in the cloud or on-prem data center. By workspace we mean an operating system, some form of storage and some amount of memory and CPU power. They appear to the user as if they are single machine in the sense of a laptop or desktop, but often there are multiple instantiations running on a single machine in the cloud. Because of this it is possible to declare things like the amount of memory, type of storage, CPU power, operating system, etc.
