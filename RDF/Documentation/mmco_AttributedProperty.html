definition
This is a sligthly lighter-weight version of reification, for use with property mmco:hasPredicate. To create a sourced statement of the form "X pred true", where pred is a boolean property, do the following: create an instance A of AttributedProperty, and add the statements "X hasAttributedProperty A" and "A hasPredicate pred". Then identify the source via "A hasSource ..." We could extend this to non-boolean properties by creating a property mmco:hasObject - this is one step closer to full reification (except that we don't specify the subject of the AttributedProperty - instead, hanging the AttirbutedProperty off the instance that would be the subject in full reification).
