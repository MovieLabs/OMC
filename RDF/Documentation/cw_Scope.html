definition
An attribute that narrows the concept represented by a CreativeWork. This has the practical effect of limiting the set of Renditions of the CreativeWork. But unlike an arbitrary VocabularyTerm, which when used as a search term can also narrow the set of Renditions satisfying the term, a Scope refers to the narrowing of the concept represented by one CreativeWork, to yield another CreativeWork.
