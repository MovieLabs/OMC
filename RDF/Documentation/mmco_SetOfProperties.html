definition
This is a mechanism for representing differences between versions, editions, etc. of a creative work. A creative work may have different values for an attribute, such as approximate length, depending on, for example, what country it is released in. To support this, we can attach different attribute sets to the creative work. For example, we could have one attribute set with hasCountry = U.S. and hasApproximateLength = 90 minutes, and another attribute set with hasCountry = Germany and hasApproximateLength = 100 minutes. Both of these attribute sets may describe the same creative work.
