definition
Asset Functional Class describes how an Asset is used in the production process. A Script is always a Script (its Functional Class), but its Structural Class might be a physical document or a digital document. Thinking of a video file as a Daily or an image as a VFX plate is more natural and useful to the participants in the production. The ontology uses this pattern, rather than directly creating subclasses of Asset, in order to support a potentially large and evolving set of different structural classes.
